# AI Usage & Guidance

## Tools Used
- ChatGPT for generating prompts and project guidance
- Antigravity for code generation.

## How AI Was Used
- I provided ChatGPT with my knowledge of the project (design rules, structure, component requirements)
- ChatGPT suggested prompts to generate code snippets, folder structures, and guidance
- I used these prompts to produce frontend and backend components in antigravity

## Constraints
- All generated code must be reviewed before integration
- Keep components modular and readable
- Follow separation of concerns
- Avoid unnecessary complexity

## Verification
- Manual code review was done for each AI-generated snippet
- Ensured alignment with project architecture and design principles
